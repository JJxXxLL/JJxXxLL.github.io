  • з¬¬дёүеұӮпјҡеҲҶи§Јдёә 4 дёӘеӯҗй—®йўҳпјҢжҜҸдёӘеӯҗй—®йўҳд»Јд»·дёә $f(n/4) = n/4$пјҢжҖ»д»Јд»· $n$гҖӮ

  • жҖ»ж—¶й—ҙеӨҚжқӮеәҰпјҡ T(n)=n+n+n+вӢҜ+n(е…ұ logвҒЎn еұӮ)T(n) = n + n + n + \dots + n \quad (\text{е…ұ } \log n \text{ еұӮ}) T(n)=nlogвҒЎnT(n) = n \log n

    еҰӮжһң $f(n) = O(n^{p - \epsilon})$пјҢе…¶дёӯ $\epsilon > 0$пјҢеҲҷпјҡ

  2. жғ…еҶө 2пјҡе№іиЎЎеўһй•ҝ
    еҰӮжһң $f(n) = \Theta(n^p)$пјҢеҲҷпјҡ

  3. жғ…еҶө 3пјҡеҲ’еҲҶ/еҗҲе№¶дё»еҜј
    еҰӮжһң $f(n) = \Omega(n^{p + \epsilon})$пјҢе…¶дёӯ $\epsilon > 0$пјҢе№¶дё”ж»Ўи¶іжӯЈеҲҷжҖ§жқЎд»¶пјҡ

    еҲҷпјҡ

дҪҝз”Ёдё»е®ҡзҗҶзҡ„жӯҘйӘӨ

  1. зЎ®е®ҡйҖ’еҪ’е…¬ејҸдёӯзҡ„еҸӮж•° $a$гҖҒ$b$ е’Ң $f(n)$пјӣ
  2. и®Ўз®— $p = \log_b a$пјӣ
  3. жҜ”иҫғ $f(n)$ е’Ң $n^p$ зҡ„еўһй•ҝзҺҮпјҡ
    • еҰӮжһң $f(n)$ еўһй•ҝеҫ—жӣҙж…ўпјҢеҲҷз”Ёжғ…еҶө 1пјӣ
    • еҰӮжһң $f(n)$ еўһй•ҝзӣёеҗҢпјҢеҲҷз”Ёжғ…еҶө 2пјӣ
    • еҰӮжһң $f(n)$ еўһй•ҝеҫ—жӣҙеҝ«пјҢеҲҷз”Ёжғ…еҶө 3гҖӮ

зӨәдҫӢ

зӨәдҫӢ 1пјҡеҪ’е№¶жҺ’еәҸ

йҖ’еҪ’е…ізі»пјҡ

  • $a = 2$пјҢ$b = 2$пјҢ$f(n) = O(n)$пјӣ
  • $p = \log_2 2 = 1$пјӣ
  • $f(n) = O(n^1)$пјҢдёӨиҖ…еўһй•ҝзҺҮзӣёеҗҢгҖӮ

з»“жһңпјҡдҪҝз”Ёжғ…еҶө 2пјҡ

зӨәдҫӢ 2пјҡзҹ©йҳөд№ҳжі•пјҲStrassenз®—жі•пјү

йҖ’еҪ’е…ізі»пјҡ

  • $a = 7$пјҢ$b = 2$пјҢ$f(n) = O(n^2)$пјӣ
  • $p = \log_2 7 \approx 2.81$пјӣ
  • $f(n) = O(n^2)$пјҢеҚі $f(n)$ еўһй•ҝжҜ” $n^p$ ж…ўгҖӮ

з»“жһңпјҡдҪҝз”Ёжғ…еҶө 1пјҡ

зӨәдҫӢ 3пјҡ$T(n) = 2T(n/2) + n^2$

йҖ’еҪ’е…ізі»пјҡ

  • $a = 2$пјҢ$b = 2$пјҢ$f(n) = O(n^2)$пјӣ
  • $p = \log_2 2 = 1$пјӣ
  • $f(n) = \Omega(n^{p + 1}) = \Omega(n^2)$пјҢдё”ж»Ўи¶іжӯЈеҲҷжҖ§жқЎд»¶гҖӮ

з»“жһңпјҡдҪҝз”Ёжғ…еҶө 3пјҡ

зӨәдҫӢ 4пјҡ$T(n) = 3T(n/3) + \log n$

йҖ’еҪ’е…ізі»пјҡ

  • $a = 3$пјҢ$b = 3$пјҢ$f(n) = O(\log n)$пјӣ
  • $p = \log_3 3 = 1$пјӣ
  • $f(n) = O(n^0)$пјҢеҚі $f(n)$ еўһй•ҝжҜ” $n^p$ ж…ўгҖӮ

з»“жһңпјҡдҪҝз”Ёжғ…еҶө 1пјҡ

дё»е®ҡзҗҶзҡ„дјҳзјәзӮ№

дјҳзӮ№пјҡ

  1. еҝ«йҖҹеҲҶжһҗж ҮеҮҶйҖ’еҪ’е…ізі»пјӣ
  2. з®ҖеҢ–еӨҚжқӮеәҰеҲҶжһҗиҝҮзЁӢгҖӮ

еұҖйҷҗжҖ§пјҡ

  1. д»…йҖӮз”ЁдәҺ $T(n) = aT(n/b) + f(n)$ еҪўејҸпјӣ
  2. дёҚйҖӮз”ЁдәҺдёҚи§„еҲҷйҖ’еҪ’е…ізі»жҲ–еҠЁжҖҒи§„еҲ’й—®йўҳгҖӮ

жҖ»з»“

дё»е®ҡзҗҶйҖҡиҝҮеҲҶжһҗйҖ’еҪ’е…¬ејҸдёӯеӯҗй—®йўҳзҡ„еўһй•ҝйҖҹзҺҮдёҺйўқеӨ–е·ҘдҪңд»Јд»·зҡ„еҜ№жҜ”пјҢиғҪеҝ«йҖҹзЎ®е®ҡйҖ’еҪ’е…¬ејҸзҡ„жёҗиҝ‘еӨҚжқӮеәҰгҖӮе®ғзү№еҲ«йҖӮз”ЁдәҺеҲҶжІ»з®—жі•пјҢеҰӮеҪ’е№¶жҺ’еәҸгҖҒеҝ«йҖҹжҺ’еәҸе’Ңзҹ©йҳөд№ҳжі•зӯүгҖӮ
