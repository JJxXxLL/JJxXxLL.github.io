з¬¬1з«  йҮҸеҢ–и®ҫи®ЎдёҺеҲҶжһҗеҹәзЎҖ

1.и®Ўз®—жңәзҡ„еҲҶзұ»зұ»еҲ«

FlynnеҲҶзұ»жі•пјҲеҹәдәҺжҢҮд»ӨжөҒе’Ңж•°жҚ®жөҒж•°йҮҸпјү

  • SISD (Single Instruction Single Data)
    • еҚ•жҢҮд»ӨеҚ•ж•°жҚ®жөҒ
    • йҖӮз”ЁдәҺеҚ•еӨ„зҗҶеҷЁзі»з»ҹпјҲUniprocessorsпјүгҖӮ
  • MISD (Multiple Instruction Single Data)
    • еӨҡжҢҮд»ӨеҚ•ж•°жҚ®жөҒ
    • зү№ж®Ҡжғ…еҶөпјҢйҖҡеёёдёҚеёёи§ҒпјҢе…·дҪ“е®һдҫӢиҫғе°‘пјҲ???пјүгҖӮ
  • SIMD (Single Instruction Multiple Data)
    • еҚ•жҢҮд»ӨеӨҡж•°жҚ®жөҒ
    • йҖӮеҗҲе№¶иЎҢеӨ„зҗҶпјҢеёёз”ЁдәҺеҗ‘йҮҸеӨ„зҗҶеҷЁе’ҢеӣҫеҪўеӨ„зҗҶгҖӮ
    • зӨәдҫӢпјҡIlliac-IVгҖҒCM-2гҖӮ
  • MIMD (Multiple Instruction Multiple Data)
    • еӨҡжҢҮд»ӨеӨҡж•°жҚ®жөҒ
    • еёёз”ЁдәҺеӨҡж ёеӨ„зҗҶеҷЁе’Ңе№¶иЎҢи®Ўз®—гҖӮ
    • зӨәдҫӢпјҡSPARCCenterгҖҒT3DгҖӮ

2.и®Ўз®—жңәзі»з»ҹз»“жһ„е®ҡд№үе’Ңи®Ўз®—жңәзҡ„и®ҫи®Ўд»»еҠЎпјҡжҢҮд»ӨйӣҶз»“жһ„жҰӮеҝөеҸҠиҰҒзҙ 

и®Ўз®—жңәзі»з»ҹз»“жһ„пјҲзҺ°д»Је®ҡд№үпјүпјҡжҳҜеңЁж»Ўи¶іеҠҹиғҪгҖҒжҖ§иғҪе’Ңд»·ж јзӣ®ж Үзҡ„жқЎд»¶дёӢпјҢи®ҫи®ЎгҖҒйҖүжӢ©е’Ңдә’иҝһзЎ¬д»¶йғЁд»¶жһ„жҲҗи®Ўз®—жңәгҖӮ

зі»з»ҹз»“жһ„иҰҶзӣ–пјҡ

вҖӢ жҢҮд»Өзі»з»ҹи®ҫи®Ў

вҖӢ з»„жҲҗпјҲOrganizationпјү:и®Ўз®—жңәи®ҫи®Ўж–№йқўзҡ„й«ҳеұӮж¬Ў:CPUеҶ…йғЁз»“жһ„гҖҒеӯҳеӮЁеҷЁгҖҒI/Oзі»з»ҹгҖҒеӨҡеӨ„зҗҶеҷЁгҖҒзҪ‘з»ң

вҖӢ зЎ¬д»¶: и®Ўз®—жңәзҡ„е…·дҪ“е®һзҺ°жҠҖжңҜ:иҜҰз»ҶйҖ»иҫ‘и®ҫи®ЎгҖҒе°ҒиЈ…гҖҒеҶ·еҚҙзі»з»ҹгҖҒжқҝзә§и®ҫи®ЎпјҢеҠҹиҖ—зӯү

ж №жҚ®иҜҫд»¶еҶ…е®№пјҢе…ідәҺжҢҮд»ӨйӣҶз»“жһ„жҰӮеҝөеҸҠиҰҒзҙ пјҢж•ҙзҗҶеҰӮдёӢйҮҚзӮ№еҶ…е®№пјҡ

жҢҮд»ӨйӣҶз»“жһ„пјҲInstruction Set Architecture, ISAпјүзҡ„жҰӮеҝө

жҢҮд»ӨйӣҶз»“жһ„пјҲISAпјүжҳҜи®Ўз®—жңәзі»з»ҹдёӯиҪҜд»¶дёҺзЎ¬д»¶зҡ„жҺҘеҸЈпјҢе®ғе®ҡд№үдәҶи®Ўз®—жңәиғҪеӨҹжү§иЎҢзҡ„жҢҮд»ӨгҖҒж•°жҚ®зұ»еһӢгҖҒеҜ„еӯҳеҷЁз»„з»ҮгҖҒеӯҳеӮЁи®ҝй—®зӯүеҶ…е®№гҖӮ
ISA зҡ„дҪңз”ЁжҳҜпјҡ

  1. дёәзЁӢеәҸе‘ҳжҸҗдҫӣдёҖдёӘзј–зЁӢи§Ҷи§’пјҡе®ҡд№үеҸҜи§Ғзҡ„жңәеҷЁиҜӯиЁҖеұһжҖ§гҖӮ
  2. зЎ¬д»¶е®һзҺ°зҡ„еҹәзЎҖпјҡзЎ¬д»¶жү§иЎҢзЁӢеәҸж“ҚдҪңж—¶йңҖдёҘж јйҒөеҫӘ ISAгҖӮ
  3. иҪҜд»¶дёҺзЎ¬д»¶зҡ„жЎҘжўҒпјҡжҢҮд»ӨйӣҶдҪҝж“ҚдҪңзі»з»ҹе’Ңеә”з”ЁзЁӢеәҸиғҪеӨҹеңЁдёҚеҗҢзЎ¬д»¶е®һзҺ°дёҠиҝҗиЎҢгҖӮ

жҢҮд»ӨйӣҶз»“жһ„зҡ„дёғдёӘиҰҒзҙ 

жҢҮд»ӨйӣҶз»“жһ„и®ҫи®ЎйңҖиҰҒиҖғиҷ‘д»ҘдёӢдёғдёӘе…ій”®иҰҒзҙ пјҡ

  1. еҜ„еӯҳеҷЁз»„з»ҮпјҲRegister Organizationпјү
    • ж—©жңҹзҙҜеҠ еҷЁз»“жһ„пјҡж“ҚдҪңж•°дҫқиө–зҙҜеҠ еҷЁпјҲеҚ•еҜ„еӯҳеҷЁпјүгҖӮ
    • зҺ°д»ЈйҖҡз”ЁеҜ„еӯҳеҷЁз»“жһ„пјҡж”ҜжҢҒеӨҡдёӘеҜ„еӯҳеҷЁпјҢжҸҗеҚҮж•°жҚ®и®ҝй—®йҖҹеәҰпјҢеҮҸе°‘еҜ№еҶ…еӯҳзҡ„дҫқиө–гҖӮ
      зӨәдҫӢпјҡ32дҪҚARMжһ¶жһ„дёӯжңү16дёӘйҖҡз”ЁеҜ„еӯҳеҷЁгҖӮ
  2. еӯҳеӮЁеҷЁи®ҝй—®пјҲMemory Accessпјү
    • е®ҡд№үж•°жҚ®еҰӮдҪ•еӯҳеӮЁе’ҢиҜ»еҸ–пјҡ
      • еӯ—иҠӮи®ҝй—®пјҲByte-level accessпјү
      • ең°еқҖеҜ№йҪҗи§„еҲҷпјҲAddress Alignmentпјүпјҡж•°жҚ®зҡ„еӯҳеӮЁең°еқҖйңҖеҜ№йҪҗеӯ—иҫ№з•ҢгҖӮ
  3. еҜ»еқҖж–№ејҸпјҲAddressing Modesпјү
    • е®ҡд№үжҢҮд»ӨеҰӮдҪ•жҢҮе®ҡж“ҚдҪңж•°зҡ„дҪҚзҪ®пјҡ
      • зӣҙжҺҘеҜ»еқҖпјҡж“ҚдҪңж•°зҡ„ең°еқҖзӣҙжҺҘеңЁжҢҮд»Өдёӯз»ҷеҮәгҖӮ
      • й—ҙжҺҘеҜ»еқҖпјҡж“ҚдҪңж•°зҡ„ең°еқҖеӯҳеӮЁеңЁеҜ„еӯҳеҷЁжҲ–еҶ…еӯҳдёӯгҖӮ
      • еҒҸз§»еҜ»еқҖпјҡеҹәең°еқҖеҠ дёҠдёҖдёӘеҒҸз§»йҮҸзЎ®е®ҡж“ҚдҪңж•°дҪҚзҪ®гҖӮ
      • з«ӢеҚіж•°еҜ»еқҖпјҡж“ҚдҪңж•°зӣҙжҺҘдҪңдёәжҢҮд»Өзҡ„дёҖйғЁеҲҶз»ҷеҮәгҖӮ
  4. ж“ҚдҪңж•°зұ»еһӢе’ҢеӨ§е°ҸпјҲOperand Types and Sizesпјү
    • е®ҡд№үж”ҜжҢҒзҡ„ж•°жҚ®зұ»еһӢе’Ңж•°жҚ®е®ҪеәҰпјҡ
      • ж•°жҚ®зұ»еһӢпјҡ8дҪҚеӯ—з¬ҰгҖҒ16дҪҚзҹӯж•ҙеһӢгҖҒ32дҪҚж•ҙеһӢгҖҒ64дҪҚжө®зӮ№ж•°зӯүгҖӮ
      • ж“ҚдҪңж•°ж•°йҮҸпјҡеҚ•ж“ҚдҪңж•°гҖҒеҸҢж“ҚдҪңж•°гҖҒдёүж“ҚдҪңж•°зӯүгҖӮ
  5. ж“ҚдҪңзұ»еһӢпјҲOperation Typesпјү
    • жҢҮд»Өж”ҜжҢҒзҡ„еҹәжң¬ж“ҚдҪңпјҢдё»иҰҒеҢ…жӢ¬пјҡ
      • ж•°жҚ®дј иҫ“пјҡеҰӮ LoadпјҲеҠ иҪҪпјүгҖҒStoreпјҲеӯҳеӮЁпјүгҖӮ
      • з®—жңҜ/йҖ»иҫ‘иҝҗз®—пјҡеҰӮеҠ жі•гҖҒеҮҸжі•гҖҒANDгҖҒORзӯүгҖӮ
      • жҺ§еҲ¶жөҒжҢҮд»ӨпјҡеҰӮи·іиҪ¬гҖҒжқЎд»¶еҲҶж”ҜгҖҒеӯҗзЁӢеәҸи°ғз”ЁдёҺиҝ”еӣһгҖӮ
  6. жҺ§еҲ¶жөҒжҢҮд»ӨпјҲControl Flow Instructionsпјү
    • е®ҡд№үзЁӢеәҸеҰӮдҪ•и·іиҪ¬е’Ңжү§иЎҢпјҡ
      • жқЎд»¶и·іиҪ¬пјҡж №жҚ®жқЎд»¶жү§иЎҢи·іиҪ¬ж“ҚдҪңгҖӮ
      • ж— жқЎд»¶и·іиҪ¬пјҡзӣҙжҺҘиҪ¬з§»зЁӢеәҸзҡ„жү§иЎҢдҪҚзҪ®гҖӮ
      • еӯҗзЁӢеәҸи°ғз”Ё/иҝ”еӣһпјҡз”ЁдәҺеҮҪж•°зҡ„и°ғз”Ёе’Ңиҝ”еӣһгҖӮ
  7. жҢҮд»Өзј–з ҒпјҲInstruction Encodingпјү
    • е®ҡд№үжҢҮд»ӨеҰӮдҪ•зј–з Ғе’ҢеӯҳеӮЁпјҡ
      • еӣәе®ҡй•ҝеәҰжҢҮд»ӨпјҡжҜҸжқЎжҢҮд»Өзҡ„й•ҝеәҰзӣёзӯүпјҢдҫҝдәҺзЎ¬д»¶е№¶иЎҢеӨ„зҗҶгҖӮ
      • еҸҜеҸҳй•ҝеәҰжҢҮд»Өпјҡж №жҚ®ж“ҚдҪңзҡ„еӨҚжқӮжҖ§пјҢжҢҮд»Өй•ҝеәҰеҸҜд»ҘеҸҳеҢ–пјҢиҠӮзңҒеӯҳеӮЁз©әй—ҙгҖӮ

жҖ»з»“

жҢҮд»ӨйӣҶз»“жһ„пјҲISAпјүжҳҜи®Ўз®—жңәзі»з»ҹзҡ„ж ёеҝғз»„жҲҗйғЁеҲҶпјҢе®ғеҶіе®ҡдәҶзЁӢеәҸдёҺзЎ¬д»¶зҡ„дәӨдә’ж–№ејҸгҖӮISAзҡ„и®ҫи®Ўдё»иҰҒеӣҙз»•еҜ„еӯҳеҷЁз»„з»ҮгҖҒеӯҳеӮЁи®ҝй—®гҖҒеҜ»еқҖж–№ејҸгҖҒж“ҚдҪңж•°зұ»еһӢгҖҒж“ҚдҪңзұ»еһӢгҖҒжҺ§еҲ¶жөҒжҢҮд»ӨеҸҠжҢҮд»Өзј–з ҒзӯүдёғдёӘиҰҒзҙ еұ•ејҖгҖӮ

еңЁзҺ°д»Ји®Ўз®—жңәи®ҫи®ЎдёӯпјҢзІҫз®ҖжҢҮд»ӨйӣҶпјҲRISCпјүе’ҢеӨҚжқӮжҢҮд»ӨйӣҶпјҲCISCпјүжҳҜдёӨз§Қдё»иҰҒзҡ„ISAи®ҫи®ЎйЈҺж јпјҡ

  • RISCпјҡжҢҮд»Өз®ҖеҚ•гҖҒжү§иЎҢеҝ«йҖҹпјҢйҖӮеҗҲзЎ¬д»¶дјҳеҢ–гҖӮ
  • CISCпјҡжҢҮд»ӨеӨҚжқӮгҖҒеҠҹиғҪејәеӨ§пјҢйҖӮеҗҲеҮҸе°‘жҢҮд»Өж•°йҮҸгҖӮ

3.е®һзҺ°жҠҖжңҜзҡ„и¶ӢеҠҝпјҡжҠҖжңҜеҸ‘еұ•зҡ„и¶ӢеҠҝ

ж‘©е°”е®ҡеҫӢпјҡйӣҶжҲҗеңЁиҠҜзүҮдёҠзҡ„жҷ¶дҪ“з®Ўж•°йҮҸжҜҸдёӨе№ҙзҝ»дёҖз•ӘпјҢжҖ§иғҪжҸҗеҚҮзҡ„еҗҢж—¶жҲҗжң¬дёӢйҷҚгҖӮ

жҖ§иғҪи¶ӢеҠҝпјҡеёҰе®Ҫж”№иҝӣдјҳдәҺж—¶е»¶

  • еёҰе®ҪпјҲBandwidthпјүпјҡеңЁз»ҷе®ҡж—¶й—ҙеҶ…е®ҢжҲҗзҡ„е·ҘдҪңжҖ»йҮҸгҖӮ
  • ж—¶е»¶пјҲLatencyпјүпјҡдёҖдёӘдәӢд»¶д»ҺејҖе§ӢеҲ°з»“жқҹжүҖйңҖзҡ„ж—¶й—ҙгҖӮ
  • з»ҸйӘҢжі•еҲҷпјҡ
    • еёҰе®ҪжҸҗеҚҮзҡ„йҖҹеәҰиҰҒдјҳдәҺж—¶е»¶зҡ„ж”№е–„йҖҹеәҰгҖӮ
  • ж”№иҝӣж–№еҗ‘пјҡ
    • еӨ„зҗҶеҷЁжҖ§иғҪжҸҗй«ҳпјҢжӣҙеӨҡе…іжіЁе№¶иЎҢеҢ–жҠҖжңҜпјҲеҰӮеӨҡж ёеӨ„зҗҶеҷЁпјүе’ҢеёҰе®Ҫжү©еұ•гҖӮ

4.йӣҶжҲҗз”өи·ҜеҠҹиҖ—зҡ„и¶ӢеҠҝпјҡеҠҹиҖ—зҡ„жҰӮеҝө
